### 원인

`src/renderer/layout.rs:2285-2300` (Issue #480 / Task #295 도입 분기):

```rust
} else if !is_tac && tbl_is_square {
    // [Issue #480] Square wrap 표는 paragraph 영역 (col_area + margin) 기준으로 정렬.
    // 이전 동작(col_area 기준)은 paragraph margin/indent 가 있는 경우 표가
    // 단 사이 갭으로 떨어지는 문제 발생 (예: 페이지 14 [A] 박스).
    let area_x = col_area.x + effective_margin;
    let area_w = (col_area.width - effective_margin - margin_right).max(0.0);
    let x = match t.common.horz_align {
        crate::model::shape::HorzAlign::Right | crate::model::shape::HorzAlign::Outside =>
            area_x + (area_w - tbl_w).max(0.0),
        crate::model::shape::HorzAlign::Center =>
            area_x + (area_w - tbl_w).max(0.0) / 2.0,
        _ => area_x,
    };
    Some(x)
}
```

이 분기는 모든 Square-wrap 표를 **무조건 문단 좌측 가장자리(`col_area.x + effective_margin`) 기준**으로 강제 배치하며 `t.common.horz_rel_to` 속성을 무시한다.

### 수식 검증

```
effective_margin = (margin_left + indent) / 2 / 75   (indent>0)
                 = (1700 + 2000) / 2 / 75
                 = 1850 / 75
                 = 24.67 px

table_x = col_area.x + effective_margin + h_offset
        = 117.17    + 24.67            + 9.44
        = 151.28 px
```

→ SVG 실측치 (셀 left = 151.28 px) 와 정확히 일치.

### 기대 동작 (HWP 표준)

`compute_table_x_position` (`src/renderer/layout/table_layout.rs:975-1005`) 의 `horz_rel_to` 분기 의미를 따라야 함:

| `horz_rel_to` | 기준 | 식 |
|---|---|---|
| **Column (단)** ← 본 케이스 | 단 좌측 | `col_area.x + h_offset` |
| Para (문단) | 문단 좌측 | `col_area.x + effective_margin + h_offset` (현재 분기 동작) |
| Page (쪽) | 본문 좌측 | `body_area.x + h_offset` |
| Paper (용지) | 용지 좌측 | `0 + h_offset` |

### 회귀 위험

Issue #480 / Task #295 는 14쪽 [A] 박스(`indent=-3800`, hanging) 케이스를 위해 도입되었음. 14쪽 [A] 도 `horz=단` 이므로 동일 분기에서 처리됨.

- 14쪽 효과: indent < 0 → effective_margin = margin_left = 11.33 px → 현재 11.33 px 우측 시프트 중
- 17쪽 효과: indent > 0 → effective_margin = (margin_left+indent)/2 = 24.67 px → 현재 24.7 px 우측 시프트 중 (사용자 보고)

본질 정정이므로 다음 영역에 회귀 가능성:

1. 14쪽 [A] (이전에 #480 으로 정정된 케이스 — 다시 정정해야 할 수 있음)
2. 다른 샘플의 Square wrap 표 (글상자, 표 모두)
3. `horz_rel_to=Para` 케이스 (현재 분기 그대로 유지되어야 함)
4. 다단/단일 단/표분할 상호작용
