# Task #22: CSS 디자인 토큰 도입 + 모바일 반응형 대응

> 수행계획서 | 2026-04-01
> Issue: [#22](https://github.com/edwardkim/rhwp/issues/22)

---

## 1. 목표

rhwp-studio의 CSS에 디자인 토큰(CSS 변수)을 도입하고, 모바일 브라우저에서 사용 가능한 반응형 레이아웃을 구현한다.

## 2. 현황 분석

| 항목 | 현재 | 목표 |
|------|------|------|
| CSS 변수 | 0개 | ~30개 토큰 |
| 하드코딩 색상 | 19종, 40줄+ | 토큰 참조 |
| 미디어 쿼리 | 0개 | 3개 브레이크포인트 |
| 터치 타겟 | 22~28px | 44px (모바일) |
| font-size | 13단계 난립 | 5단계 체계 |

## 3. 구현 계획 (4단계)

### 단계 1: 색상 토큰 도입

`base.css`의 `:root`에 색상 토큰을 선언하고, 주요 파일부터 점진 적용한다.

- 브랜드 색상: `--color-primary`, `--color-primary-dark`, `--color-accent-bg` 등
- 표면 색상: `--color-surface`, `--color-bg`, `--color-bg-light`
- 테두리: `--color-border`, `--color-border-light`, `--color-border-dark`
- 텍스트: `--color-text`, `--color-text-secondary`, `--color-text-muted`
- 상호작용: `--color-hover-bg`, `--color-active-bg`, `--color-focus-border`
- 그림자: `--shadow-dropdown`, `--shadow-dialog`

적용 범위: 22개 CSS 파일 중 사용 빈도 높은 파일부터 전환

### 단계 2: 타이포그래피 + 간격 토큰

- 폰트 크기: `--font-size-xs`(10px) ~ `--font-size-xl`(16px) 5단계
- 폰트 패밀리: `--font-family-ui`, `--font-family-mono`
- 간격: `--space-1`(2px) ~ `--space-8`(16px)
- 크기: `--radius-sm/md/lg`, `--control-height-sm/md/lg`

### 단계 3: 반응형 브레이크포인트 + 레이아웃

브레이크포인트 정의:
- 모바일: `~767px`
- 태블릿: `768px~1023px`
- 데스크톱: `1024px~` (현재 레이아웃 유지)

모바일 레이아웃 변경:
- 메뉴바 → 햄버거 메뉴
- 도구 상자 → 하단 플로팅 또는 접기
- 서식 도구 모음 → 가로 스크롤 또는 접기
- 대화상자 → 풀스크린 시트
- 상태 표시줄 → 최소 정보만 표시
- 편집 영역 → 전체 화면, 핀치 줌

터치 대응:
- 버튼/아이콘 터치 타겟 44px 최소 보장
- 스와이프 제스처 (메뉴 열기/닫기)
- 가상 키보드 출현 시 레이아웃 조정

### 단계 4: 기존 코드 점진 전환 + 검증

- 수정 기회 발생 시 해당 파일의 하드코딩 값을 토큰으로 교체
- E2E 테스트에 모바일 뷰포트 시나리오 추가
- 주요 디바이스 크기별 스크린샷 비교

## 4. 원칙

- **경량 토큰**: 토큰 수 30개 이내, 과도한 추상화 금지
- **점진적 마이그레이션**: 일괄 전환 금지, 새 코드부터 적용
- **도구 미도입**: Sass/PostCSS 불필요, CSS 변수만 사용
- **다크 모드 미대응**: 현시점 불필요 (토큰 체계로 향후 전환 용이)
- **데스크톱 우선**: 기존 데스크톱 레이아웃을 깨지 않으면서 모바일 추가

## 5. 검증 방법

- 데스크톱: 기존 E2E 테스트 52 assertions 전부 통과
- 모바일: Chrome DevTools 디바이스 모드 + CDP E2E 뷰포트 변경 테스트
- 시각적: 스크린샷 비교 (데스크톱/태블릿/모바일)

## 6. 산출물

- `base.css`: 토큰 선언 (:root)
- 22개 CSS 파일: 토큰 참조로 전환 (점진적)
- 반응형 미디어 쿼리: 주요 컴포넌트별
- E2E 모바일 TC 추가
- CLAUDE.md: CSS 접두어 규칙에 미문서화 접두어 추가
