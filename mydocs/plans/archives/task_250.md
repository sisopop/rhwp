# Task 250 수행 계획서: 직선 도형 편집 개선

## 현황
- 직선 그리기: 드래그로 생성 가능하나 수평/수직 제약 미동작
- 직선 선택: 시작/끝 2개 핸들 표시되나 드래그로 편집 불가
- Shift+직선: 이전에 구현되었으나 현재 동작 안 함 (regression)

## 해결 사안

### 1단계: 수평/수직 직선 그리기 복원
- Shift+드래그 시 수평(dy=0) 또는 수직(dx=0) 제약
- 45도 단위 스냅도 지원 (Shift 유지 시)
- regression 원인 파악 및 수정

### 2단계: 끝점 마커 드래그 — 각도/길이 조절
- 시작점(sw) 핸들 드래그: 시작점 이동 → 각도 변경
- 끝점(ne) 핸들 드래그: 끝점 이동 → 각도 변경
- 선의 bbox(horizontal_offset, vertical_offset, width, height)와 start/end 좌표 동기 갱신
- Shift+핸들 드래그: 수평/수직/45도 제약

### 3단계: 길이 조절
- 끝점 핸들을 선 방향으로 드래그 → 길이만 변경 (각도 유지)
- 또는 자유 드래그로 각도+길이 동시 변경 (2단계와 통합)
