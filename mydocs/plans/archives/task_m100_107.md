# 수행계획서 — Task #107: Chrome 확장 호버 카드 클릭 HWP 열기 구현

---

## 1. 문제 정의

`showHoverCard()` 함수에서 생성한 호버 카드에 클릭 이벤트가 등록되어 있지 않아, '클릭하여 rhwp로 열기' 문구가 표시되어도 실제로 동작하지 않는다.

badge 클릭 이벤트는 이미 구현되어 있으며 (`chrome.runtime.sendMessage({ type: 'open-hwp', url: anchor.href })`), 호버 카드도 동일한 방식으로 처리하면 된다.

---

## 2. 수정 방향

card 요소에 click 이벤트 핸들러를 추가하여 badge와 동일하게 `open-hwp` 메시지를 전송한다.

추가로 CSS에 `cursor: pointer`를 적용하여 클릭 가능함을 시각적으로 표시한다.

---

## 3. 수정 범위

| 파일 | 수정 내용 |
|------|-----------|
| `rhwp-chrome/content-script.js` | card에 click 이벤트 핸들러 추가 |
| `rhwp-chrome/content-script.css` | `.rhwp-hover-card`에 `cursor: pointer` 추가 |

---

## 4. 단계 계획

**단계 1 (단일)**: 클릭 이벤트 추가 + CSS cursor 적용 + 빌드 확인
