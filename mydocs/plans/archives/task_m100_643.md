# Task #643: 페이지 분할 드리프트 정정 (pi=80 마지막 줄 다음 페이지 누락) — 수행계획서

## 목표

`samples/2022년 국립국어원 업무계획.hwp` 6페이지 마지막 문단(pi=80) 2번째 줄(' 및 점자 해당 분야 전문인력 확보 어려움')이 다음 페이지로 부당하게 분리되는 문제를 해결한다. HWP 원본과 동일하게 같은 페이지에 배치되어야 한다.

## 본질 진단

### 증상
- pi=80은 2줄 문단 (62자, line=160%, lh=1500HU=20px)
- HWP vpos 기준: line1 bottom = 931.2px ≤ body 933.5px → 2.3px 여유로 fits
- 우리 렌더러: page used=906.7px (hwp_used=899.2px 대비 **+7.5px 드리프트**)
- 잔여 26.8px < line1 line_advance 32px → page break 발생

### Root Cause 가설
`src/renderer/pagination/engine.rs:864` 의 fit 판정이 누적 `current_height` (sb + line_advance + sa) 기반.
페이지 상단 표(pi=61, wrap=TopAndBottom, tac=true) 부근에서 드리프트 시작 의심:
- HWP vpos diff (pi=61→pi=62): 4503 HU = 60.0px
- 우리 표 보고 높이: 43.5px → 16.5px 갭 처리 부재 의심

### 영향 범위
- 단일 케이스로 보이나, drift 누적은 일반 패턴이므로 다른 문서에서도 잠재 이슈 가능
- 회귀 검증 필수 (164 fixture 1,614 페이지)
