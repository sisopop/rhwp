## 3. 가설

표 셀 내부 paragraph가 셀 너비에 맞춰 자동 줄바꿈될 때, 2번째 LINE_SEG 이후의 `vpos`가 누적되지 않아 같은 위치에 그려짐. 가능한 원인:

- (A) `line_height = 0` 으로 처리되어 줄간격 0
- (B) vpos accumulator가 표 셀 컨텍스트에서 누락
- (C) LINE_SEG vpos가 첫 번째 줄의 값으로 고정
