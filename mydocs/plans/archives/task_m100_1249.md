## 2. 확인 목표

이번 작업은 신규 구현보다 **현재 devel에서 #1249 증상이 이미 해소되었는지**를 확인하는 데 초점을 둔다.

확인 기준:

1. HWPX parser가 `<hp:subList textDirection="VERTICALALL">`을 `text_box.list_attr` bit 0~2에 반영하는지 확인
2. HWPX 글상자 세로쓰기 회귀 테스트가 통과하는지 확인
3. HWPX와 HWP5 변환본 SVG에서 세로쓰기 rotate transform 수가 동일한지 확인
4. 신규 소스 수정 필요 여부 판단
