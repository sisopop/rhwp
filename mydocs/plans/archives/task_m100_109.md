# 수행계획서 — Task #109

**이슈**: [#109](https://github.com/edwardkim/rhwp/issues/109)
**타이틀**: Chrome 확장 호버카드 썸네일 UX 개선 — 클릭 문서 열기 + hover 애니메이션
**마일스톤**: M100
**작성일**: 2026-04-12
**브랜치**: `local/task109`

---

## 현상

1. 썸네일 위에서 cursor가 `default`로 표시 — 클릭 가능한지 사용자가 인식 불가
2. 썸네일 hover 시 시각적 피드백 없음
3. (카드 전체 click 이벤트는 이미 연결되어 있으므로 클릭 자체는 동작함)

## 원인

`content-script.css`에서 `.rhwp-hover-thumb img`에 `cursor: pointer` 및 hover 효과가 없다.

## 수정 범위

`rhwp-chrome/content-script.css` — `.rhwp-hover-thumb` 및 `.rhwp-hover-thumb img` 스타일

## 수정 방향

1. `.rhwp-hover-thumb img`에 `cursor: pointer` 추가 — 썸네일이 클릭 가능함을 암시
2. `.rhwp-hover-thumb` hover 시 애니메이션 효과:
   - `transform: scale(1.03)` — 살짝 확대
   - `brightness` filter — 약간 밝게
   - `transition` — 부드러운 전환
