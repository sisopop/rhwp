# Task m100 #1155: 장평/자간 단축키 지원

## 1. 목표

한컴 편집기에 익숙한 사용자가 자주 쓰는 장평/자간 1% 증감 단축키를 rhwp-studio에 구현한다.

대상 이슈:

```text
#1155 rhwp-studio: 단축키로 장평/자간 1% 증감 지원
```

대표 사용 시나리오:

```text
2줄로 배치된 문장을 1줄로 만들기 위해 텍스트를 블럭 선택한 뒤,
Shift+Alt+J 또는 한글 입력 모드의 Shift+Alt+ㅓ 로 장평을 1%씩 줄인다.
```

## 2. 한컴 호환 단축키

| 기능 | Windows/Linux | macOS | 한글 입력 모드 키 |
|---|---|---|---|
| 장평 1% 줄이기 | Shift+Alt+J | Shift+Option+J | Shift+Alt+ㅓ |
| 장평 1% 늘리기 | Shift+Alt+K | Shift+Option+K | Shift+Alt+ㅏ |
| 자간 1% 줄이기 | Shift+Alt+N | Shift+Option+N | Shift+Alt+ㅜ |
| 자간 1% 늘리기 | Shift+Alt+W | Shift+Option+W | Shift+Alt+ㅈ |

macOS의 Option 키는 브라우저 `KeyboardEvent.altKey`로 들어오므로, 내부 매칭은 기존 `alt` modifier를 사용한다.

## 3. 현재 구현 지점

현재 rhwp-studio는 단축키와 서식 명령이 다음 구조로 연결된다.

```text
rhwp-studio/src/command/shortcut-map.ts
  - defaultShortcuts
  - matchShortcut()

rhwp-studio/src/command/commands/format.ts
  - format:font-size-increase/decrease
  - format:line-spacing-increase/decrease

rhwp-studio/src/engine/input-handler.ts
  - getCharProperties()
  - applyCharPropsToRange()
  - adjustFontSize()
```

글자 속성에는 이미 HWP char shape의 장평/자간 필드가 존재한다.

```text
CharProperties.ratios?: number[]    // 장평
CharProperties.spacings?: number[]  // 자간
```

따라서 새 저장 포맷을 만들지 않고 기존 char format command 경로로 처리한다.

## 4. 구현 계획

### 4.1 단축키 매핑

`shortcut-map.ts`에 다음 커맨드를 추가한다.

```text
format:char-ratio-decrease
format:char-ratio-increase
format:char-spacing-decrease
format:char-spacing-increase
```

각 커맨드는 영문 키와 한글 입력 모드 키를 모두 매핑한다.

```text
J / ㅓ
K / ㅏ
N / ㅜ
W / ㅈ
```

IME 입력 중 `event.key`가 `Process` 또는 예상 문자와 다르게 들어오는 브라우저/OS 조합을 위해
`event.code` 기반 매칭도 보강한다.

```text
KeyJ
KeyK
KeyN
KeyW
```

### 4.2 장평/자간 적용

`format.ts`에 새 format command를 추가하고, input handler에 글자 속성 증감 헬퍼를 추가한다.

처리 규칙:

```text
장평:
  current = ratios[0] 또는 100
  delta = -1 또는 +1
  clamp = 50..200
  적용값 = Array(7).fill(next)

자간:
  current = spacings[0] 또는 0
  delta = -1 또는 +1
  clamp = -50..50
  적용값 = Array(7).fill(next)
```

블럭 선택이 없으면 기존 글꼴 크기 증감과 동일하게 아무 작업도 하지 않는다.
이번 이슈의 핵심 사용성은 선택 범위에 대한 즉시 적용이다.

### 4.3 검증

최소 검증:

```text
cd rhwp-studio
npm test
npm run build
```

추가 테스트:

```text
rhwp-studio/tests/shortcut-map.test.ts
```

테스트할 항목:

```text
Shift+Alt+J/K/N/W 매핑
Shift+Alt+ㅓ/ㅏ/ㅜ/ㅈ 매핑
IME Process + event.code(KeyJ/KeyK/KeyN/KeyW) 매핑
macOS Option에 해당하는 altKey 조합 매핑
```

## 5. 완료 기준

```text
1. 블럭 선택 후 장평 줄이기/늘리기 단축키가 동작한다.
2. 블럭 선택 후 자간 줄이기/늘리기 단축키가 동작한다.
3. 한글 입력 모드에서도 동일 단축키가 동작한다.
4. macOS Option 조합에서도 동일 단축키가 동작한다.
5. 기존 서식 단축키, 문단 이동, 브라우저 충돌 회피 로직이 회귀하지 않는다.
```
