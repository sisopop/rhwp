## 1. 배경

`samples/hwpx/form-002.hwpx` 의 SVG 출력에서 외부 표 row 19 의 셀[73] (rs=1, cs=27) 안의 inner 표 (`연구개발계획서 제출시…`, p[28]) 가 page 2 상단에 와야 할 위치임에도 page 1 하단에 표출됨.

PDF/HWP 기준에서는 같은 inner 표가 page 2 에 존재함.

## 2. 근본 원인

`src/renderer/layout/table_layout.rs::compute_cell_line_ranges()` 의 `has_table_in_para` 분기 (라인 ~2052-2086):

- 가시성 결정 시 `limit_remaining` 검사가 누락
- 셀 분할에서 `content_limit > 0` 일 때, limit 초과 후 등장하는 중첩 표 포함 문단이 **항상 표시(`(0, line_count)`)** 됨
- 일반 문단은 line 루프 내 `if limit_remaining <= 0.0 { break; }` 로 정상 차단됨 → 비대칭

## 3. 목표

- 셀 분할(`split_end_content_limit > 0`) 시 중첩 표 포함 문단이 limit 을 넘으면 atomic 단위로 다음 페이지로 미룸
- form-002.hwpx 1~2 페이지 PDF 와 일치
- 기존 회귀 없음 (다른 샘플 문서 SVG 비교)
