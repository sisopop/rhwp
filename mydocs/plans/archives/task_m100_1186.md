# Task M100 #1186 — cargo clippy all-targets baseline 정리

- 이슈: https://github.com/edwardkim/rhwp/issues/1186
- 작성일: 2026-06-03
- 브랜치: `local/task_m100_1186`
- 성격: 기능 변경 없는 lint baseline 정리

## 배경

`cargo clippy --all-targets -- -D warnings` 가 `devel` baseline 에서 실패한다.
일반 CI 범위인 `cargo clippy -- -D warnings` 는 통과하지만, 테스트 타깃과 예제를 포함하면
기존 테스트/진단 코드의 lint 가 error 로 승격된다.

## 목표

- `cargo clippy --all-targets -- -D warnings` 무경고 통과
- 기능 동작 변경 없이 clippy 지적만 정리
- 향후 `--all-targets` CI 도입 가능 상태로 baseline 확보

## 범위

- 테스트 코드의 `get(0)` → `first()` 변환
- `Box::new(Type::default())` → `Box::default()` 변환
- `vec![]` 불필요 사용 제거
- `len() > 0`, `len() >= 1` 표현 정리
- doc comment list indentation 정리
- 상수 assertion, format literal, iterator 표현 등 clippy 경고 정리
- `items_after_test_module` 해소를 위한 테스트 모듈 전 item 이동

## 비범위

- 런타임 동작 변경
- 렌더링/저장 알고리즘 변경
- public API 의미 변경

## 검증 계획

1. `cargo clippy --all-targets -- -D warnings`
2. `cargo fmt --all --check`
3. `cargo test --tests`
