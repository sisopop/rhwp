# Task #86: HWP 썸네일 자동 추출 + data-hwp-thumbnail 연동 — 수행 계획서

## 수행 목표

HWP 파일 내장 미리보기 이미지(PrvImage)를 추출하여, 웹 관리자가 최소 코드로 방문자에게 HWP 문서 썸네일을 제공할 수 있도록 한다.

## 범위

1. WASM API: `extractThumbnail()` — CFB에서 PrvImage만 빠르게 추출
2. CLI: `rhwp thumbnail` 명령 — PNG/base64 출력
3. Chrome 확장 연동: `data-hwp="true"` 링크 호버 시 썸네일 자동 표시

## 핵심 설계 원칙

- **웹 관리자 최소 코드**: `data-hwp="true"` 한 줄이면 충분
- **서버 불필요**: 클라이언트 WASM으로 PrvImage 추출 완결
- **빠른 응답**: 전체 문서 파싱 없이 CFB 스트림만 부분 읽기

## 기존 구현 현황

- `src/model/document.rs`: `Preview`, `PreviewImage`, `PreviewImageFormat` 구조체
- `src/parser/cfb_reader.rs`: `read_preview_image()`, `read_preview_text()`
- `src/parser/mod.rs`: `extract_preview()` 함수
- WASM API: 미노출 상태
