# Task 243 수행 계획서: 도형 속성 편집 UI

## 현재 상태

기존 `PicturePropsDialog`에 Shape용 6개 탭이 이미 구현되어 있음:
- 기본(크기/위치/회전/대칭), 여백/캡션, 선(테두리/화살표/모서리), 채우기(단색/그라디언트), 글상자(여백/정렬), 그림자

## 구현 항목

### 완료
1. **`line` 타입 전용 탭 목록** — `LINE_TAB_NAMES = ['기본', '여백/캡션', '선', '그림자']`
   - 채우기/글상자 탭 제외 (직선에 불필요)
2. **`line` 타입 속성 저장** — `setShapeProperties` 호출 경로 추가
3. **속성 초기값 로드** — `line` 타입도 `getShapeProperties` 사용

### 확인 필요
- 직선 선택 → 우클릭 또는 도구상자 > 개체 속성 → 속성 대화상자 열림
- "선" 탭에서 테두리 색/두께/스타일/화살표 변경 후 적용
- "기본" 탭에서 크기/위치 변경 후 적용
