## 2. 왜 Stage5 universal clamp 와 다른가 (회귀 회피 가설)

- Stage5 clamp/proxy 는 **전 미주 para** 의 advance 를 일괄 override → 텍스트 제목/꼬리 19건 과압축.
- v2 는 **지점·조건을 극히 좁힘**: paragraph_layout trailing 계산에서 아래 3조건 **동시 충족** 시에만
  phantom 마커를 정상 line-spacing 으로 치환:
  1. `para_is_treat_as_char_equation_only(para)` — 빈 텍스트 + TAC 수식 only (제목/꼬리=텍스트 제외)
  2. `line_spacing_hu ≈ self.endnote_between_notes_hu.get()`(±tol) — trailing 이 between-notes 마커
  3. same-endnote successor(=미주 내부, between-notes 경계 아님 → 마커가 phantom)
- pi=1127(텍스트)·pi=1129(그림, ls=452)·19건 질문 제목/꼬리(텍스트)는 조건1·2 불충족 → 무영향(가설).
