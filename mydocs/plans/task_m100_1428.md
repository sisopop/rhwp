## 문제 인식

#1282 후속 시각 검증 중 rhwp-studio의 개체 속성 대화상자가 한컴오피스와 다르게 동작하는 문제가 확인되었다.

1. `기본` 탭의 너비/높이 입력이 항상 원본 비율을 유지한다.
2. `비율 유지` 체크 옵션이 없어 사용자가 너비와 높이를 독립적으로 입력할 수 없다.
3. 개체 속성 및 일부 모달이 바깥 영역 클릭만으로 닫힌다.

`비율 유지`는 HWP/HWPX 파일 포맷에 저장되는 속성이 아니라, 대화상자에서 한쪽 크기 입력 시 다른 축을 자동 보정할지 결정하는 사용자 UI 설정으로 취급한다. 다크모드와 같이 `rhwp-settings`에 기억하되, HWP5/HWPX parser/serializer 포맷 매핑은 이번 이슈의 직접 수정 범위가 아니다.

## 조사 결과

- `rhwp-studio/src/ui/picture-props-dialog.ts` 기본 탭은 `widthInput`과 `heightInput`의 `input` 이벤트에서 항상 `originalWidth/originalHeight` 비율로 반대 축을 갱신한다.
- 같은 파일의 그림 탭에는 `가로 세로 같은 비율 유지` 체크가 이미 있으나, 기본 탭의 `크기` 영역에는 한컴오피스의 `비율 유지`에 해당하는 체크박스가 없다.
- `picture-props-dialog.ts`와 공통 `ModalDialog`는 overlay 클릭 시 닫히는 경로가 존재한다.

## 목표

1. 개체 속성 `기본` 탭의 크기 영역에 `비율 유지` 체크박스를 추가한다.
2. 체크 ON일 때만 너비/높이 반대 축 자동 보정을 수행한다.
3. 체크 OFF일 때는 너비와 높이를 독립적으로 입력할 수 있게 한다.
4. `비율 유지` 선택값은 다크모드처럼 사용자 설정에 저장해 다음 대화상자 열기에서도 유지한다.
5. 개체 속성 및 일반 모달은 외부 클릭으로 닫히지 않고 명시적 버튼으로만 닫히게 한다.
6. 파일 포맷 저장 속성으로 오해하지 않도록 기술 문서 또는 작업 문서에 범위를 명확히 남긴다.

## 검증 기준

- 개체 속성 기본 탭에 `비율 유지` 옵션이 표시된다.
- `비율 유지` OFF 상태에서 너비 변경 시 높이가 바뀌지 않고, 높이 변경 시 너비가 바뀌지 않는다.
- `비율 유지` ON 상태에서 한쪽 값을 바꾸면 다른 쪽 값이 기존 비율로 갱신된다.
- `비율 유지` ON/OFF 선택값이 `rhwp-settings`에 저장되고 다음 대화상자 열기에도 유지된다.
- 개체 속성 모달과 공통 모달이 외부 클릭만으로 닫히지 않는다.
- 관련 E2E 또는 단위 수준 UI 검증을 추가한다.
