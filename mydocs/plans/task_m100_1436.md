## 문제 인식

한컴오피스에서는 개체 속성의 `크기 고정`을 켜면 단순히 너비/높이 값을 저장하는 데 그치지 않고, UI와 직접 조작 경로 전체가 잠긴다.

- 너비/높이 입력 비활성화
- 회전각 입력 비활성화
- 기울이기 입력 비활성화
- 선택 핸들에서 리사이즈/회전/기울이기 조작 차단
- 저장/취소/닫기 같은 명시적 조작 외 모달 종료 차단

rhwp에는 파일 포맷 속성 자체는 존재한다. `CommonObjAttr.size_protect`가 HWP5 `CTRL_HEADER` bit 20 및 HWPX `hp:sz@protect`로 파싱/직렬화된다. 하지만 Studio 속성 JSON과 UI/명령 처리에 이 값이 충분히 노출되지 않아, `크기 고정` 상태가 한컴처럼 강제되지 않는다.

## 조사 결과

- `mydocs/tech/한글문서파일형식_5.0_revision1.3.md` 표 70에는 `bit 20`이 `VertRelTo이 para일 때 크기 보호 여부`로 이미 등록되어 있다.
- `src/model/shape.rs`: `CommonObjAttr.size_protect` 필드가 존재한다.
- `src/parser/control/shape.rs`: HWP5 attr bit 20을 `size_protect`로 파싱한다.
- `src/parser/hwpx/section.rs`: HWPX `protect` 속성을 `size_protect`로 파싱한다.
- `src/document_core/commands/object_ops.rs`: `common_obj_attr_to_json`과 picture JSON 포맷에 `sizeProtect`가 아직 포함되어 있지 않다.
- `rhwp-studio/src/ui/picture-props-dialog.ts`: `sizeFixedCheck` UI는 있으나 속성 로드/저장/비활성화 연동이 부족하고, 회전 입력은 로드 시 다시 활성화된다.
- `rhwp-studio/src/engine/input-handler-picture.ts`: 키보드/드래그 리사이즈와 회전 드래그 경로가 크기 고정 여부를 별도로 검사하지 않는다.

추가 확인: 새로 제공된 `ta-pic-001-r-쪽영역안제한-크기고정.hwpx`를 기존 `ta-pic-001-r-쪽영역안제한.hwpx`와 비교하면 첫 번째 그림의 `<hp:sz ... protect="0">`가 `protect="1"`로 바뀐 것이 핵심 차이다. 이는 기술문서의 bit 20 설명 및 HWPX `hp:sz@protect` 의미와 일치한다. 별도 fixture로 등록된 코드 참조는 없었으므로 신규 크기고정 샘플 파일은 커밋 대상에서 제외했다.

## 목표

1. `size_protect`를 Studio 속성 JSON에 `sizeProtect`로 노출한다.
2. 개체 속성 대화상자가 `sizeProtect`를 읽고 저장하도록 한다.
3. `sizeProtect`가 켜진 개체는 너비/높이/회전/기울기 관련 UI를 한컴처럼 비활성화한다.
4. 리사이즈/회전/기울이기 직접 조작 경로에서도 `sizeProtect`를 방어적으로 차단한다.
5. 개체 속성 및 설정 모달의 외부 클릭 닫힘 회귀 여부를 함께 확인하고, 남은 경로가 있으면 정리한다.

## 제외 범위

- `비율 유지`는 #1428에서 다룬 사용자 UI 설정이며, 이번 이슈의 파일 포맷 속성인 `크기 고정`과 분리한다.
- 한컴 UI에 없는 새 파일 포맷 필드는 추가하지 않는다.
- 드롭다운/메뉴처럼 일시 UI로서 바깥 클릭 닫힘이 자연스러운 컴포넌트는 모달 정책 범위에서 제외한다.

## 검증 기준

- `크기 고정`이 켜진 그림/도형을 열면 개체 속성 대화상자의 너비/높이/회전각/기울기 입력이 비활성화된다.
- `크기 고정` 토글 저장 후 HWPX `protect` 및 HWP5 bit 20 라운드트립이 유지된다.
- 선택 핸들 또는 키보드 리사이즈가 크기 고정 개체에 적용되지 않는다.
- 회전/기울기 조작이 크기 고정 개체에 적용되지 않는다.
- 개체 속성 모달 바깥 문서 영역 클릭으로 모달이 닫히지 않는다.
- 기존 개체 속성 저장/취소, #1428 `비율 유지`, #1282 표 안 그림 리사이즈 동작은 회귀하지 않는다.
