- 변경된 Stage31 목적은 한컴 도움말 기준 미주 기능 구현이다. 미주는 본문 위치에 번호를 남기고 미주 내용은 문서 끝 또는 구역 끝에 모이며, 미주 모양은 번호 서식/장식 문자/구분선/여백/번호 매기기/위치를 구역 속성으로 설정한다.
- 한컴 도움말 기준 미주 실행 직후 커서는 미주 내용 위치로 이동하고 `주석` 탭이 나타나며, 본문 복귀는 `<Shift+Esc>` 또는 `닫기`로 수행한다.
- 구현 결과: 새 빈 문서에서 미주 삽입 시 `Control::Endnote`와 미주 내용 `AutoNumberType::Endnote` 문단을 만들고, 본문/미주 내용에는 기본 `1)` 마커가 렌더된다.
- 미주 모양 API는 `SectionDef.endnote_shape`를 대상으로 번호 서식, 앞/뒤 장식 문자, 구분선, 구분선 위/아래, 미주 사이, 번호 매기기, 문서/구역 끝 위치를 조회/적용한다.
- 미주 모양 적용 시 기존 미주의 본문 번호와 미주 내용 번호도 함께 재번호/재동기화하여 앞 장식 문자, 뒤 장식 문자, 시작 번호가 즉시 반영된다.
- rhwp-studio는 미주/각주 삽입 직후 기존 note editing 경로로 진입하고, 한컴처럼 `주석` 전용 도구상자에 `각주`, `미주`, `각주/미주 모양`, `닫기` 버튼을 표시한다.
