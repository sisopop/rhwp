# Task 1129 Stage 26 - 고배율 격자 점 가시성 보정

## 사용자 지적

- 현재 격자 점이 2배 확대 상태에서 보이지 않는다.

## 원인 가설

- Stage 25에서 점 지름을 1px 기준으로 낮추면서 `0.5px → 0.55px`의 매우 좁은 투명 전환 구간을 둔 radial-gradient를 사용했다.
- 브라우저/디바이스 픽셀 정렬에 따라 0.5px 반경 점이 안티앨리어싱으로 흐려지거나 사라질 수 있다.

## 수정 계획

- 점 격자는 1px 중심 픽셀이 확실히 칠해지도록 hard stop 형태로 바꾼다.
- 격자 간격 계산은 유지한다.

## 수정 내용

- 점 격자 배경 생성에 현재 보기 배율(`zoom`)을 전달한다.
- 점 반경은 `max(0.5px, zoom * 0.5px)`로 계산해 2배 확대 시 반경 1px로 표시되게 했다.
- 투명 전환 구간을 제거해 0.5px 안티앨리어싱 때문에 점이 사라지는 경우를 줄였다.
