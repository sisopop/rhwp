## 검증 계획

macOS 로컬 검증은 `mydocs/manual/dev_environment_guide.md`의 "macOS 로컬 빌드/테스트 검증" 절을 따른다.

1. `cargo build --release`
2. `cargo test --release --lib`
3. `cargo test --profile release-test --tests`
4. `cargo fmt --check`
5. 위 명령이 모두 통과하면 `wasm-pack build --target web --out-dir pkg`

## 실패 원인과 조치

- `issue_1082`의 2024 sep20/20 가드는 Task #1363 이후 overflow 0px급을 기대하는 `5px` 타이트 바운드였다.
- Task #1293 stage113/stage122에서는 같은 샘플의 p19/p20/p22 잔여를 공식 미주 간격식 불일치가 아니라
  문28 본문/그림/수식 continuation 높이 차이의 tail/cascade로 분류했다.
- 현재 최종 sweep 기준 잔여와 테스트 기대값이 충돌하므로, `tests/issue_1082_endnote_multicolumn_drift.rs`의
  sep20/20 전용 바운드를 `SEP2020_RESIDUAL_LIMIT_PX = 40.0`으로 조정했다.
  - 현재 잔여 `29.3px`에는 여유를 둔다.
  - 옛 수백 px 회귀와 종전 약 `50px` 과충전은 계속 잡는다.
