  - `cur=924.29`, `avail=1001.56`, `gap=26.45px`
  - `default_gap=true`, `compact_gap=true`, `visible_sep=true`, `has_rewind=true`
  - `large_head_outside=true`, `allow_default_title=false`, `advance_new=true`
- 기존 `allow_default_question_title_tail`의 large-below rewind branch는 같은 케이스를 다루지만
  `st.current_height < available * 0.92` 상한 때문에 `924.29 / 1001.56 ~= 0.923`인 p20을 제외했다.

## 수정

- `src/renderer/typeset.rs`
  - 기본 `미주 사이` + visible separator + 큰 `구분선 아래` + 저장 vpos rewind + 첫 단 하단에서
