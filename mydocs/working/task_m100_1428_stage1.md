# Task 1428 Stage 1

## 목표

- 개체 속성 `기본` 탭에 `비율 유지` 체크박스를 추가한다.
- `비율 유지` ON/OFF에 따라 너비/높이 자동 보정 여부를 분리한다.
- 개체 속성 및 일반 모달이 외부 클릭만으로 닫히지 않게 한다.

## 구현 방침

- `비율 유지`는 파일 포맷 저장 속성이 아니라 다크모드처럼 `rhwp-settings`에 저장되는 사용자 UI 설정으로 처리한다.
- 기존 rhwp-studio 동작과의 급격한 차이를 줄이기 위해 기본값은 ON으로 둔다.
- 사용자가 ON/OFF를 변경하면 즉시 설정에 저장하고 다음 개체 속성 대화상자에서도 같은 선택값을 사용한다.
- 모달 닫기는 확인, 취소, 닫기 버튼, Escape처럼 명시적 조작으로만 수행한다.

## 검증 계획

- 개체 속성 기본 탭에서 OFF일 때 너비/높이 독립 입력을 확인한다.
- ON일 때 한쪽 입력으로 다른 축이 기존 비율에 맞게 갱신되는지 확인한다.
- ON/OFF 선택값이 `rhwp-settings`에 저장되는지 확인한다.
- overlay 클릭으로 모달이 닫히지 않는지 E2E 또는 수동 검증 자료로 확인한다.

## 구현 결과

- `user-settings.ts`에 `dialog.picturePropsKeepRatio` 사용자 설정을 추가했다.
- 개체 속성 기본 탭의 `비율 유지` 체크박스가 `rhwp-settings` 저장값을 읽고, 변경 시 즉시 저장하도록 연결했다.
- `비율 유지` OFF 상태에서는 너비/높이 입력 상호 갱신을 하지 않고, ON 상태에서만 기존 비율로 반대 축을 갱신한다.
- 개체 속성 및 주요 모달의 overlay 클릭 닫힘 경로를 제거했다.
