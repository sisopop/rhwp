**컨트리뷰터 환경별 권위 분리** (2026-05-07 PR #670 갱신):

## Windows + 한컴 편집기 환경
- **1차 정답지**: 한컴 2010 / 2020 / 2022 **편집기** 직접 출력 (시각 판정)
- **보조**: `pdf/` (한글 2022) / `pdf-2020/` / `pdf-2010/` 의 PDF 출력

## macOS / Linux 환경 (한컴 편집기 미접근)
- **1차 정답지**: `pdf/` (한글 **2022**) 또는 `pdf-2020/` (한글 **2020**) PDF
- **등급 미달**: `pdf-2010/` (한글 2010 PDF) — 보조 자료

> 한컴 편집기는 Windows 전용이므로, 편집기에 접근할 수 없는 맥/리눅스 컨트리뷰터에게는 한글 2020/2022 PDF가 권위 정답지 역할을 한다. 이는 rhwp의 외부 컨트리뷰터 다양성 인프라이며 `project_dtp_identity` 의 정체성을 뒷받침한다.

## 모든 환경 공통 — 정답지 아님
- 한컴 뷰어 출력 (편집기 아님)
- macOS 인쇄 다이얼로그를 통한 PDF (한컴 외부 경로)
- 한컴 외부 변환 도구 (hwp2pdf 등)
- HWP5 v2024 변환본 등 한컴 변환 산출물 (비교 보조 자료)
- 한컴 2018 등 미확인 버전

## 적용
1. PR 검토 시 "PDF 와 일치" 검증 결과:
   - 한컴 편집기 (Windows) → 작업지시자 정답지와 비교 가능성 높음
   - 한글 2020/2022 PDF (`pdf/` 또는 `pdf-2020/`) → 맥/리눅스 컨트리뷰터에게 정답지
   - 한글 2010 PDF / 한컴 뷰어 / macOS 인쇄 → 보조 자료, 재검증 필수
2. 컨트리뷰터의 PDF 출력 방법 확인 + 한컴 버전 명시 권장
3. 작업지시자 환경의 정확한 버전 / OS 명시 권장 (commit / 보고서 / CHANGELOG)
